## Курс ASP.NET Core Web API микросервисы (GeekBrains)  
  
### Репозиторий предназначен для домашних заданий  
  
В рамках курса реализованы 3 приложения:  
  
**Metrics Agent** - собирает метрики с сервера на котором запущен, 
хранит их в базе данных и отдает при запросе по HTTP.  
  
**Metric Manager** - собирает через HTTP запросы метрики с одного или нескольких запущенных агентов метрик, 
хранит их в базе данных и отдает при запросе по HTTP.  
  
Запущенные агент и менеджер и их API можно посмотреть по адресам:  
**[http://metricsagent.verm-v.ru](http://metricsagent.verm-v.ru)**  
**[http://metricsmanager.verm-v.ru](http://metricsmanager.verm-v.ru)**  
(реализовано через swagger)  
  
**Metrics Manager Client** - WPF приложение - визуализирует метрики, которые получает из менеджера метрик через HTTP запросы 
(приложение находится в дальнейшей разработке)  
  
для проектов **Metrics Agent** и **Metric Manager** созданы проекты юнит тестов (с использованием моков)
